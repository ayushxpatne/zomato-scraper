Zomato Scraper
The Zomato Scraper is a web scraping tool developed using Python and Selenium to collect restaurant data from Zomato.

Created primarily for learning purposes, it allows users to enhance their skills in web scraping and automation. Users can specify the city they want to scrape data for and adjust the tool to fetch a desired amount of results.

With options to gather additional restaurant details and export the data in various formats, this project serves as both a practical data collection tool and an educational resource for mastering web scraping techniques.
